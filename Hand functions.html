Segmentation¶

Colour extracting¶

First algorithm that comes in "Contact" - segmentation algorithm. There are few functions which deal with it. First of them is extracting image by colour.

Colour_extract(img, colour1L, colour1R, colour2L, colour2R)

This function makes extracting image by colour and return a mask. First arg is image to do extract in, and then boundary parametrs of colour to range in. As default it search in red-colour range.

Clear noize¶

The second algorithm helps first to clear all things, that accidentally came in the image

Clear_noize(img)

The function takes image and do contour search. After that it removes all the contours except the biggest one.

Extracting hand gesture¶

And the main algorithm - extract gesture. It uses that two functions upper and return img containing mask of gesture

extract_gesture(img, L1, R1, L2, R2)

It also needs parametrs to range in for colour.

Data Base¶

Static gesture creating¶

This functions can help easily create a huge dataset for several minute with your notebook and a camera.

Create_static(path, end, l1, r1, l2, r2, imgsize = 224)

Path - the place where images will be saved. End - the number saved images after which program will be finished automatically, imgsize - the shape of image and other parametrs the same as in functions upper.

Dinamic gesture creating¶

The process of creating dinamic getures is about the same as in static except few features. The set of images is saving as one senquence if one directory.

Create_dinamic(path, end, l1, r1, l2, r2, imgsize = 50)

Path - the place where images will be saved. End - the number saved images after which program will be finished automatically, imgsize - the shape of image and other parametrs the same as in functions upper.

Safe Dinamic Algorithm¶

Create gesture¶

The function returns the sum of few images as one summar image.

Create_gesture(mas, imgsize)

Mas - list of images to plus up, imgsize - the size of resulting image.
